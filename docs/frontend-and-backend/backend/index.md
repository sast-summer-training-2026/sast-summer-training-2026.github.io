# 后端

后端，不只是写接口。可靠的服务也不能靠纯 vibe coding。AI 时代不再需要逐行手敲代码，但软件工程的思想内核没有变，这些 insight 仍然是驾驭 AI 工具的前提，而不是可以被跳过的章节。

本次分享将从零梳理后端开发的核心脉络：从 API 路由与 RESTful 设计，到中间件如何串联请求链路；从「给定业务需求怎么设计数据模型」，到缓存如何帮服务扛住高并发。同时也会涉及开发规范、测试与 CI/CD 流水线，以及 Docker 容器化部署的实际操作演示。不讲空中楼阁，每一步都落在你未来真正会写（或者指挥 Agent 写）的代码上。本次分享完全支持零基础，带上你对「一个网站背后在发生什么」的好奇心就够了。
