# 科研分享（一）

科研是许多同学进入清华后既向往又感到陌生的一件事：它值得投入吗？从哪里开始？和课内学习有什么不同？本次讲座将从主讲人的经历出发，完整梳理从萌生兴趣到深入科研的全过程，为大一大二的同学呈现一份可操作的科研入门指南。

讲座将首先回答"要不要做科研"这个问题。我们会聊聊选择科研的理由，也会还原一段真实的本科低年级科研经历。希望通过这些真实细节，帮助大家建立对科研的合理预期。

在此基础上，讲座将系统介绍科研起步的三个关键环节：进组前需要做哪些准备、如何选择合适的课题组与导师、如何逐步确定自己的研究方向。进组之后，高效读论文的方法、与学长和老师的沟通技巧、课内学习与科研时间的平衡策略，都是新人最常遇到的难题，主讲人将结合自己初入领域时的探索经历，给出具体可行的建议。

对于同学们普遍关心的发展路径问题，讲座还将专门讨论保研与出国两条道路：两者各自的评价标准是什么、需要在什么时间节点做出选择、大一大二阶段可以做哪些让两个选项都保持打开的准备。主讲人将结合自己在国外做科研和海外暑期科研的经历，分享暑研申请、推荐信积累与申请季规划的实用经验。最后，讲座将以对科研的整体思考与个人规划收尾，欢迎带着你的困惑前来交流。
