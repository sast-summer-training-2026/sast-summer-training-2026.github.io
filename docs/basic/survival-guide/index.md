# CS 生存指南

本讲面向初入计算机专业，对计算机技术了解不多的小白。本讲会通过介绍信息检索的基本方式和常见渠道，为各位计算机初学者展现利用计算机和网络解决问题的基本过程和要素。同时，本讲通过介绍最常见的若干计算机技术（如 Linux 和 Git），为各位计算机初学者在未来的科研和工作快速上手工程与实验环境做好准备。

注：本讲整体内容偏基础，不适合有一定计算机知识的同学参加。
